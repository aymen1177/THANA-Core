ما تم إنجازه

البنية الأساسية

- تثبيت Laravel وإعداد المشروع.
- إعداد Git وربط المشروع مع GitHub.
- إنشاء قاعدة البيانات.

إدارة النظام

- المستخدمون.
- المجالات (Domains).
- التصنيفات (Categories).
- الخدمات (Services).
- مقدمو الخدمات (Service Providers).
- خدمات مقدمي الخدمات (Provider Services).

نظام الطلبات

- إنشاء الطلب من طرف الزبون.
- عرض الطلبات حسب نوع المستخدم.
- تحديد السعر من طرف مقدم الخدمة.
- قبول أو رفض العرض من طرف الزبون.
- فصل لوحة التحكم حسب نوع الحساب:
  - مدير.
  - مقدم خدمة.
  - زبون.

الاختبارات

- تم اختبار دورة الطلب كاملة بنجاح:
  1. الزبون ينشئ الطلب.
  2. مقدم الخدمة يرى الطلب.
  3. مقدم الخدمة يحدد السعر.
  4. الزبون يرى السعر.
  5. الزبون يقبل أو يرفض العرض.

Git

- تم حفظ نسخة مستقرة على GitHub.
